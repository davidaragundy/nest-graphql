# Dev

1. Clonar el proyecto
2. Copiar el `.env.template` y renombarlo a `.env`
3. Ejecutar

```
npm i
```

4. Levantar la imagen (docker desktop needed)

```
docker-compose up -d
```

5. Levantar Nest

```
npm run start:dev
```
